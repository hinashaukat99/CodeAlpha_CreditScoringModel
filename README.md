# CodeAlpha_CreditScoringModel

## Objective
The objective of this project is to predict an individual's creditworthiness using past financial and demographic data. The model classifies applicants as good or bad credit risks using machine learning techniques.

---

## Dataset
The dataset used contains financial and personal attributes such as:
- Age
- Credit amount
- Account status
- Employment history
- Loan purpose
- Savings status
- Credit history

---

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

---

## Machine Learning Models
The following classification models were implemented:
- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

---

## Project Workflow
1. Data loading  
2. Data preprocessing  
3. Handling missing values  
4. Encoding categorical variables  
5. Feature engineering  
6. Train-test split  
7. Feature scaling  
8. Model training  
9. Model evaluation  

---

## Evaluation Metrics
The models were evaluated using:
- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC score

---

## Results
- Logistic Regression achieved an accuracy of approximately 81% with balanced performance.
- Decision Tree showed lower performance compared to other models.
- Random Forest achieved the best performance with a ROC-AUC score of approximately 0.82, making it the most effective model for credit risk prediction.

---

## Conclusion
This project demonstrates a complete machine learning pipeline for credit scoring, including data preprocessing, feature engineering, model training, and evaluation. The Random Forest model performed best in terms of ROC-AUC score, indicating strong classification capability.

Future improvements can include hyperparameter tuning, handling class imbalance, and deploying the model as a web application.
